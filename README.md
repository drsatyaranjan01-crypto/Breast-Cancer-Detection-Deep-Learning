# 🩺 Breast Cancer Detection using Deep Learning

A Deep Learning-based Breast Cancer Detection System built using TensorFlow/Keras on the Wisconsin Breast Cancer Dataset.

## 🚀 Features
- Binary Classification (Benign / Malignant)
- Data Preprocessing & Feature Scaling
- Artificial Neural Network (ANN)
- TensorFlow/Keras Implementation
- 96.5% Test Accuracy

## 🛠 Tech Stack
- Python
- NumPy
- Pandas
- Scikit-Learn
- TensorFlow
- Keras
- Matplotlib

## 📊 Results
- Test Accuracy: **96.5%**
- Loss Visualization
- Accuracy Visualization

## 📂 Dataset
Wisconsin Breast Cancer Dataset (Scikit-Learn)
